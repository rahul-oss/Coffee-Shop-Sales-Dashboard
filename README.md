# Coffee-Shop-Sales-Dashboard
This dashboard provides a complete sales performance overview for a coffee shop business using interactive visualizations and business KPIs.
☕ Coffee Shop Sales Dashboard

An interactive Power BI dashboard designed to analyze coffee shop sales performance and customer purchasing behavior.

📊 Dashboard Features
Total Revenue & Quantity KPIs
Month-over-Month Revenue Trend
Peak Hour Revenue Analysis
Product Category Performance
Store-wise Revenue Comparison
Revenue by Weekday vs Weekend
Average Order Value Metrics
🛠 Tools Used
Power BI
Power Query
DAX
Data Modeling
Data Cleaning
📈 Business Insights

This dashboard helps identify:

Best-selling products
Peak customer hours
High-performing stores
Revenue trends over time
Customer purchasing patterns
